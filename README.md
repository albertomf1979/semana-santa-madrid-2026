# Semana Santa Madrid 2026

Aplicación web gratuita con el programa completo de la Semana Santa de Madrid 2026 (27 de marzo – 5 de abril).

## Funcionalidades

- **Procesiones**: 21 procesiones organizadas por día, con hora, iglesia, mapa del recorrido y enlace a Google Maps
- **Programa Religioso**: Celebraciones litúrgicas en la Catedral de la Almudena, El Retiro y Domingo de Resurrección
- **Programa Musical**: Conciertos, ciclo de órgano, encuentros de bandas y eventos culturales
- **Accesibilidad**: Tipografía grande (Helvetica), contraste WCAG 2.1, áreas táctiles mínimas de 44×44px

## Datos

Toda la información proviene del programa oficial del Ayuntamiento de Madrid (esmadrid.com).

## Tecnología

- React 18 + Vite
- Desplegado en Netlify

## Desarrollo local

```bash
npm install
npm run dev
```
